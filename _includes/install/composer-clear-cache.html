packages.magento.comにて紹介した変更にはMagetoパッケージにて意図しないエラーを起こすものが含まれていました。

これは2015年の8/13から8/17の間にcomposer create-projectを使ってMagentoプロジェクトを作成した人のみ影響があります。

あるユーザーがインストール中に発生したエラーについて現象をGitHub にて報告しました。

[UnexpectedValueException]
The checksum verification of the file failed (downloaded from https://packages.magento.com/_packages/magento_framework-1.0.0-beta.zip)

原因はpackages.magento.com とローカルの Composer cacheを比べた時のパッケージのコンフリクトです。解決するにはComposer cacheを以下のようにクリアしてください:

  1. Log in to your Magento server as a user with permissions to run composer in the Magento installation directory.
    1. MagentoサーバーにMagentoインストレーションディレクトリにてcomposerのパーミッションを持つユーザーでログインしてください
    2. cd <your Magento install dir>
    3. composer clear-cacheを入力
